# Button Dropdowns

## `bs:btn:dropdown`

	<div class="btn-group">
  		<button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
    		 <span class="caret"></span>
		</button>
	  	<ul class="dropdown-menu" role="menu">
    		<li><a href="#"></a></li>
		</ul>
	</div>
